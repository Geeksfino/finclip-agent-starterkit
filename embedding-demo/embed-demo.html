FinClip Chat Widget Integration Guide

Learn how to embed the FinClip chat widget on any website

Integration Guide

Adding the FinClip chat widget to your website is simple. There are two methods:

Method 1: Simple Embedding

Include our script with the data-finclip-chat attribute for automatic initialization, and don't forget to add the stylesheet:

<!-- Add the stylesheet in the head section -->
<link rel="stylesheet" href="https://your-domain.com/finclip-chat.css">

<!-- Simple embedding with default settings -->
<script src="https://your-domain.com/finclip-chat.js" data-finclip-chat></script>

Method 2: Advanced Configuration

For more control, initialize the widget manually with custom options:

<!-- Add the stylesheet in the head section -->
<link rel="stylesheet" href="https://your-domain.com/finclip-chat.css">

<!-- Load the script -->
<script src="https://your-domain.com/finclip-chat.js"></script>

<!-- Initialize with custom options -->
<script>
  window.initFinClipChat({
    buttonLabel: "Chat with Support",
    initialOpen: false,
    suggestions: ["How do I get started?", "What are your pricing plans?"],
    suggestionsLabel: "Frequently Asked Questions",
    apiUrl: "https://api.your-domain.com",
    streamingUrl: "https://streaming.your-domain.com"
  });
</script>

Configuration Options

  • buttonLabel: Text shown on the chat button
  • initialOpen: Whether to open the chat window on page load
  • suggestions: Array of suggested questions to show users
  • suggestionsLabel: Header text for the suggestions section
  • apiUrl: URL for the chat API endpoint
  • streamingUrl: URL for the streaming response endpoint

Required Files

Make sure to include both the JavaScript and CSS files for the widget to function properly:

  • JavaScript: finclip-chat-embed.iife.js - Contains the widget functionality
  • CSS: style.css - Contains the styling for the widget

The CSS file is essential for the widget to display correctly. Without it, the chat button and interface will not be properly styled.
